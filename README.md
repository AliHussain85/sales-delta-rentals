# sales-delta-rentals

Sales deal tracker for Delta Rentals Dubai. Single-page app that loads WhatsApp click leads from Supabase and posts closed deals to a webhook.

## Setup

Open `custom-tracker.html` in a browser or serve it from any static host.

## Stack

- Supabase (`whatsapp_clicks` table)
- Deal-closed webhook at `api.deltarentalsdubai.com`
